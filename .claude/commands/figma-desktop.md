Обнови предыдущий код, добавив адаптивность для десктопной версии макета.

Алгоритм:
1. Проанализируй десктопный макет через get_design_context
2. Сравни структуру с мобильной версией
3. Добавь md: префиксы для десктопных изменений:
   - Изменение направления flex: добавь md:flex-row к flex-col
   - Скрытые элементы: hidden md:block
   - Размеры текста: text-sm md:text-lg
   - Отступы: p-4 md:p-8

Важно: НЕ дублируй HTML структуру. Используй одни теги с адаптивными классами.

Если DOM радикально отличается, используй условный рендеринг:
```jsx
<div className="md:hidden">{/* Mobile version */}</div>
<div className="hidden md:block">{/* Desktop version */}</div>
```
