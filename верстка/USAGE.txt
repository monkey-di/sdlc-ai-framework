┌─────────────────────────────────────────────────────────────────┐
│ GEMINI CLI                                                      │
└─────────────────────────────────────────────────────────────────┘

Запустите в директории проекта:
  $ gemini

Команды (копируйте одну строку):

  /figma:init
  /figma:mobile
  /figma:desktop
  /figma:refactor src/components/Header.tsx
  /figma:html
  /figma:qa


┌─────────────────────────────────────────────────────────────────┐
│ CLAUDE CODE                                                     │
└─────────────────────────────────────────────────────────────────┘

В любом диалоге с Claude Code:

  /figma-mobile
  /figma-desktop
  /figma-refactor
  /figma-html
  /figma-qa


┌─────────────────────────────────────────────────────────────────┐
│ ТИПОВОЙ WORKFLOW (Gemini CLI)                                  │
└─────────────────────────────────────────────────────────────────┘

1. Запустите Gemini:
   $ gemini

2. Выделите МОБИЛЬНЫЙ фрейм в Figma, затем:
   /figma:mobile

3. НЕ закрывая диалог, выделите ДЕСКТОПНЫЙ фрейм в Figma:
   /figma:desktop

4. Скопируйте сгенерированный код


┌─────────────────────────────────────────────────────────────────┐
│ ТИПОВОЙ WORKFLOW (Claude Code)                                 │
└─────────────────────────────────────────────────────────────────┘

1. Выделите МОБИЛЬНЫЙ фрейм в Figma

2. В Claude Code напишите:
   /figma-mobile

3. НЕ закрывая диалог, выделите ДЕСКТОПНЫЙ фрейм в Figma

4. В том же диалоге:
   /figma-desktop

5. Скопируйте сгенерированный код


┌─────────────────────────────────────────────────────────────────┐
│ ОБНОВЛЕНИЕ СУЩЕСТВУЮЩЕГО ФАЙЛА                                  │
└─────────────────────────────────────────────────────────────────┘

Gemini CLI:
  $ gemini
  /figma:refactor src/components/Button.tsx

Claude Code:
  /figma-refactor
  → Укажите путь к файлу когда спросит


┌─────────────────────────────────────────────────────────────────┐
│ ЧИСТЫЙ HTML (БЕЗ REACT)                                         │
└─────────────────────────────────────────────────────────────────┘

Gemini CLI:
  $ gemini
  /figma:html

Claude Code:
  /figma-html


┌─────────────────────────────────────────────────────────────────┐
│ ПРОВЕРКА КАЧЕСТВА                                               │
└─────────────────────────────────────────────────────────────────┘

После генерации кода:

Gemini CLI:
  /figma:qa

Claude Code:
  /figma-qa


┌─────────────────────────────────────────────────────────────────┐
│ ВАЖНЫЕ ЗАМЕЧАНИЯ                                                │
└─────────────────────────────────────────────────────────────────┘

⚠️  НЕ закрывайте диалог между /figma:mobile и /figma:desktop
    Они работают последовательно в одном контексте!

⚠️  Выделяйте элемент в Figma ПЕРЕД запуском команды

⚠️  Для рефакторинга обязательно укажите путь к файлу

✅  Отступы в Figma кратные 4px → лучший результат
    (8px, 16px, 24px вместо 15px, 19px)

✅  Настройте Variables в Figma для правильных цветов
    (bg-primary вместо bg-[#3B82F6])
